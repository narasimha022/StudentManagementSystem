Student Management System

Overview

A Java based Student Management System developed using JDBC and a database. This application helps manage student records efficiently.

Features

- Add Student
- View Student Details
- Update Student Information
- Delete Student Records
- Database Connectivity using JDBC

Technologies Used

- Java
- JDBC
- Eclipse IDE
- MySQL

Project Structure

- src/ - Source code
- dao/ - Data Access Layer
- model/ - Student model classes
- database/ - Database connection classes
